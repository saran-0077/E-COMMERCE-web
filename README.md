# 🛒 MyStore - Advanced React E-Commerce App

A professional React application with a full shopping experience, Admin Dashboard, and Analytics.

## 🚀 Features
- **Product Catalog:** Fetched from FakeStoreAPI with caching.
- **Admin Dashboard:** Real-time sales analytics using Recharts.
- **Order History:** Persistent storage for past orders.
- **Accessibility:** ARIA-compliant UI for keyboard navigation.
- **Responsive Design:** Works on Mobile, Tablet, and Desktop.

## 🛠️ Tech Stack
- **Frontend:** React 18, React Router v6
- **State Management:** Hooks (useState, useEffect)
- **Charts:** Recharts
- **Testing:** Jest & React Testing Library
